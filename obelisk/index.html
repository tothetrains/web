Obelisk

Obelisk is planned to be the main bot for the To The Trains Discord server. Features from Dyno and Statbot will be adopted over time, eventually resulting in the phase out of these additional bots. Obelisk will serve as the main platform for server moderation, engagement, and oversight. During development the source code will be proprietary - an open source release is tabled for when the majority of planned features are in working order.

Obelisk is programmed in Python with the use of py-cord and dislash.py.

Help

CommandArgumentsDescription

(*: required)

Your Data

Obelisk can react to when a user joins or leaves and fetch the entire server member list, utilising the 'members' intent therefor. Obelisk cannot read the content of messages and can only interpret commands and their respective arguments.

Obelisk collects and stores a database of various statistics:

  • Quantity of messages sent per user
  • Quantity of users invited per user
  • Quantity of reactions per user
  • Quantity of achievements unlocked per user
  • Duration of server membership per user
  • If you are boosting/have boosted the server
